Цель работы

Изучение классических методов шифрования перестановкой (маршрутное шифрование, шифрование решетками, таблица Виженера) и реализация одного из алгоритмов программно.

Теоретическая часть:

В рамках работы были рассмотрены три основных метода:

Маршрутное шифрование (метод Виета): Открытый текст записывается в таблицу по строкам. Криптограмма получается выписыванием букв по столбцам, порядок которых определяется алфавитным порядком букв ключевого слова (пароля).

Шифрование с помощью решеток (метод Флейснера): Используется специальный трафарет, повороты которого заполняют квадратную таблицу, после чего символы выписываются по столбцам согласно ключу.

Таблица Виженера: Символы текста циклически сдвигаются по алфавиту на значение, определяемое цифровым эквивалентом букв ключа.

Практическая реализация:

В ходе работы была проанализирована и протестирована программа на языке Python (ЛАБ 2 Инормационный Безопасности.py), реализующая маршрутное шифрование (столбцовую перестановку).

Алгоритм работы программы:

  1. Подготовка текста: Из исходной строки удаляются пробелы.

  2. Создание матрицы: Текст разбивается на блоки по n символов (столбцов), которые становятся строками матрицы. Количество строк (m) определяется автоматически.

  3. Заполнение последней строки: Если последняя строка заполнена не полностью (длина текста не кратна n), она дополняется случайными символами из предопределенного алфавита.

  4. Добавление ключа: Строка с ключевым словом (key_word) вставляется нулевой строкой в начало матрицы.

  5. Сортировка столбцов: Определяется порядок столбцов для вывода. Индексы столбцов сортируются на основе лексикографического порядка букв в ключевом слове (как в описании метода Виета).

  6. Формирование результата: Криптограмма собирается путем обхода отсортированных столбцов сверху вниз (от строки с ключом до последней строки данных).

Пример работы:
Входные данные, указанные в коде:

  • Текст: "нельзя недооценивать противник ава" (после удаления пробелов)
  • Ключ: "пароль"
  • Параметры: n=6, m=5

Программа должна выдать результат, близкий к примеру в методическом пособии, однако точный вывод может отличаться из-за случайного заполнения последней строки и возможных отличий в кодировке алфавита, использованного в коде.

Выводы:

В ходе лабораторной работы был изучен принцип маршрутного шифрования. Представленная программа корректно реализует алгоритм столбцовой перестановки с использованием ключевого слова. Программа автоматически адаптирует текст под заданный размер матрицы и учитывает алфавитный порядок букв ключа для перестановки столбцов, что полностью соответствует классическому описанию метода Виета.
