# <u>Цель работы</u>

Реализовать алгоритм шифрования и дешифрования текста методом
гаммирования с использованием конечной гаммы (ключа) и операции сложения
по модулю (XOR и модуль 33 для русского алфавита).

**<u>Теоретическая часть:</u>**

- Исходный текст: **ПРИКАЗ**

- Ключ (гамма): **ГАММА**

- Алфавит: заглавные русские буквы (33 символа, включая Ё)

- Методы:

  - Побитовое XOR (универсальное, для любых символов)

  - Сложение номеров букв по модулю 33 (для русского алфавита)

# **3. Реализация**

Программа состоит из двух частей:

**3.1. Шифрование с помощью XOR (универсальное)**

Функции `encrypt` и `decrypt` выполняют побитовое сложение каждого символа
текста с соответствующим символом ключа (циклически).\
Результат:

**4. Результаты**

- При использовании XOR текст зашифровывается в нечитаемые символы
  (бинарные коды), что соответствует классическому потоковому
  шифрованию.

- При использовании алфавитного сложения по модулю 33 получен
  шифротекст **«УСХЧБК»**, что полностью совпадает с примером в методических
  указаниях.

**5. Выводы**

- Реализованы два варианта гаммирования: побитовое XOR и алфавитное
  сложение по модулю 33.

- Алгоритм обратим: повторное применение ключа восстанавливает
  исходный текст.

- Метод гаммирования с конечным ключом прост в реализации, но его
  криптостойкость зависит от длины и случайности гаммы.
