Теоретическая часть:

Шифр Цезаря — каждый символ открытого текста заменяется символом, находящимся на фиксированное число позиций k правее в алфавите. Математическая модель:

E(i) = (i + k) mod 26

где i — номер буквы в алфавите (A=0, B=1, …, Z=25), k — ключ.

Шифр Атбаш — зеркальная замена: первая буква алфавита меняется с последней, вторая — с предпоследней и т.д.

E(i) = 25 - i

Программная реализация

Инструменты: Python 3, модуль string.
Алфавит: заглавные латинские буквы A-Z (для Атбаш добавлен пробел).

3.1 Функция шифра Цезаря

import string

alphabet = list(string.ascii_uppercase)

def cesar_cipher(k, text):
result = ""
for letter in text:
idx = alphabet.index(letter)
result += alphabet[(idx + k) % 26]
return result

3.2 Функция шифра Атбаш

alphabet = list(string.ascii_uppercase) + [" "]

def atbash_cipher(text):
indexes = [alphabet.index(ch) for ch in text]
reversed_alphabet = list(reversed(alphabet))
return "".join(reversed_alphabet[i] for i in indexes)

Исправления, внесённые в исходный код:

  • Убрано глобальное изменение алфавита (reversed_alphabet создаётся локально).

  • Добавлен оператор return в функцию atbash_cipher.
