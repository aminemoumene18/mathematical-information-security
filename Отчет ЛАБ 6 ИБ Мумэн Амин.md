**2. Реализованные алгоритмы**

**2.1. ρ-метод Полларда**

*   **Вход:** составное число n, параметр c (начальное значение), функция f(x) = (x² + 5) mod n.
*   **Принцип работы:**
    *   Генерируется псевдослучайная последовательность с помощью функции f.
    *   Используется метод поиска цикла («черепаха и заяц») для нахождения коллизий.
    *   Вычисляется НОД(|a-b|, n). Если 1 < d < n, то d — нетривиальный делитель.
*   **Пример:** Для n = 1359331 и c = 1 найден делитель 1181, что совпадает с примером в методичке.

**2.2. Метод квадратов Ферма**

*   **Вход:** нечётное число n.
*   **Принцип работы:**
    *   Ищется такое s ≥ √n, что s² - n является полным квадратом (t²).
    *   Тогда n = s² - t² = (s - t)(s + t).
*   **Пример:** Для n = 65 найдены разложения:
    *   65 = 5·13 → s=9, t=4 → 65 = 9² - 4².
    *   65 = 1·65 → s=33, t=32 → 65 = 33² - 32².
