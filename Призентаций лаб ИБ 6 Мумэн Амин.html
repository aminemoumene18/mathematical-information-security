Цель работы :  
Реализовать алгоритмы факторизации составных чисел  
1. ρ-метод Полларда (Pollard's rho algorithm);  
2. Метод квадратов Ферма (разложение через разность квадратов).  
2. Реализованные алгоритмы  
2.1. ρ-метод Полларда  
 Вход: составное число n, параметр c (начальное значение), функция f(x)  
= (x² + 5) mod n.  
 Принцип работы:  
o Генерируется псевдослучайная последовательность с помощью  
функции f.  
o Используется метод поиска цикла («черепаха и заяц») для  
нахождения коллизий.  
o Вычисляется НОД(|a-b|, n). Если 1 < d < n, то d — нетривиальный  
делитель.  
 Пример: Для n = 1359331 и c = 1 найден делитель 1181, что совпадает с  
примером в методичке.  
2.2. Метод квадратов Ферма  
 Вход: нечётное число n.  
 Принцип работы:  
o Ищется такое s ≥ √n, что s² - n является полным квадратом (t²).  
o Тогда n = s² - t² = (s - t)(s + t).  
 Пример: Для n = 65 найдены разложения:  
o 65 = 5·13 → s=9, t=4 → 65 = 9² - 4².  
o 65 = 1·65 → s=33, t=32 → 65 = 33² - 32².  
3. Код программы :  
Прилагается файл ЛАБ 6 Информационный безопасности.py, содержащий две  
функции:  
 pollard_method(n, c): реализует ρ-метод Полларда.  
 factors(n): реализует метод квадратов Ферма.
