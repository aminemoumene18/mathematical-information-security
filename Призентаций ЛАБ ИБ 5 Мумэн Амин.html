Цель работы

Реализовать программно три вероятностных алгоритма проверки целых чисел на простоту: тест Ферма, тест Соловэя-Штрассена и тест Миллера-Рабина.

Теоретическая часть:

В работе используются следующие математические основы и алгоритмы:

Тест Ферма: Основан на малой теореме Ферма. Если для числа n существует такое a, что an−1 ≠ 1 (mod n), то n — составное.

Символ Якоби: Обобщение символа Лежандра, используемый в тесте Соловэя-Штрассена. Вычисляется рекурсивно на основе свойств квадратичных вычетов.

Тест Соловэя-Штрассена: Использует критерий Эйлера: для простого n должно выполняться a(n−1)/2 ≡ ( a/n ) (mod n), где ( a/n ) — символ Якоби.

Тест Миллера-Рабина: Более сложный тест, основанный на свойствах разложения n−1 = 2s · r и проверке корней из единицы по модулю n.

Анализ результатов:

В ходе тестирования были проверены как простые числа (7, 17, 29), так и составные (15, 21), а также число Кармайкла (561).

  • Тест Ферма: Успешно определил составные числа 15 и 21. Однако для числа Кармайкла 561 тест может ошибочно дать результат "вероятно простое", так как для него выполняется сравнение Ферма для всех оснований a, взаимно простых с n. Это демонстрирует основной недостаток теста.

  • Тест Соловэя-Штрассена: Более надежен. Число Кармайкла 561 не проходит этот тест для большинства оснований, так как равенство a(n−1)/2 ≡ ( a/n ) (mod n) для него не всегда выполняется.

  • Тест Миллера-Рабина: Показал наилучшие результаты. Даже для чисел Кармайкла он с высокой вероятностью определяет их как составные.

Вывод:

В ходе лабораторной работы были изучены и программно реализованы три вероятностных алгоритма проверки чисел на простоту.

  • Тест Ферма прост в реализации, но уязвим для чисел Кармайкла.
  • Тест Соловэя-Штрассена сложнее, но надежнее теста Ферма.
  • Тест Миллера-Рабина является наиболее предпочтительным, так как он эффективен, широко используется на практике и не имеет "слепых зон", подобных числам Кармайкла.

Для повышения достоверности результатов на практике необходимо проводить несколько итераций каждого теста с разными случайными основаниями a.
