находящимся на фиксированное число позиций kk правее в алфавите.

Шифр Атбаш — зеркальная замена: первая буква алфавита меняется с

3.1 Функция шифра Цезаря  
python  
import string  
alphabet = list(string.ascii_uppercase)  
def cesar_cipher(k, text):  
result = ""  
for letter in text:  
idx = alphabet.index(letter)  
result += alphabet[(idx + k) % 26]  
return result  
3.2 Функция шифра Атбаш  
python  
alphabet = list(string.ascii_uppercase) + [" "]  
def atbash_cipher(text):  
indexes = [alphabet.index(ch) for ch in text]  
reversed_alphabet = list(reversed(alphabet))  
return "".join(reversed_alphabet[i] for i in indexes)
