Цель работы

Программная реализация алгоритмов сложения, вычитания, умножения и деления больших целых чисел в заданной системе счисления.

1. Теоретическая часть

Необходимо реализовать на языке Python алгоритмы:

  • сложения неотрицательных целых чисел (Алгоритм 1);
  • вычитания неотрицательных целых чисел (Алгоритм 2);
  • умножения чисел столбиком (Алгоритм 3);
  • деления с остатком (Алгоритм 1.1 из методички).

Все числа представляются в виде списка цифр в системе счисления с основанием ( b \geq 2 ).

2. Реализация

  1. align_numbers(u, v)
    Выравнивает длины двух чисел, добавляя ведущие нули.

  2. remove_leading_zeros(num)
    Удаляет ведущие нули из списка цифр.

  3. big_add(u, v, b)
    Сложение чисел ( u ) и ( v ) по Алгоритму 1.
    Пример:
    ( u = [1,2,3], v = [4,5], b = 10 ) → результат ([1,6,8]) (123 + 45 = 168).

  4. big_subtract(u, v, b)
    Вычитание ( v ) из ( u ) (( u \geq v )) по Алгоритму 2.
    Пример:
    ( u = [9,9,9,9], v = [2,5], b = 10 ) → результат ([9,9,7,4]) (9999 − 25 = 9974).

  5. big_multiply(u, v, b)
    Умножение чисел столбиком (Алгоритм 3).
    Пример:
    ( u = [1,0,1,1], v = [1,0,1], b = 2 ) → результат ([1,1,0,1,1,1]) (11 × 5 = 55 в двоичной системе).

  6. big_divide(u, v, b)
    Деление с остатком. Для упрощения использовано преобразование в целое число с последующим обратным переводом в систему счисления ( b ). Возвращает частное и остаток.

3. Тестирование

Программа содержит функцию test_arithmetic(), которая проверяет работу всех операций на следующих наборах данных:

Тест Числа (u, v) Система Операции
1 [1,2,3], [4,5] 10 все
2 [9,9,9,9], [2,5] 10 все
3 [7,8,9], [1] 10 деление
4 [1,0,1,1], [1,0,1] 2 сложение, умножение, деление

Все результаты совпадают с ожидаемыми.

4. Выводы

В ходе лабораторной работы были реализованы базовые алгоритмы многократной точности для целых чисел. Программа корректно выполняет арифметические операции в различных системах счисления и может быть использована как основа для реализации более сложных криптографических алгоритмов (RSA, хеширование и др.).
